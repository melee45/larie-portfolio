# Larie Amimirog — Portfolio

This is a simple static portfolio site for Larie Amimirog. It was scaffolded from an existing HTML/CSS file and is ready to deploy.

Files:

- index.html — the site source (static HTML + CSS).

Quick local preview:

Using Python 3 (no install required):

```powershell
python -m http.server 8000
# then open http://localhost:8000 in your browser
```

Deploy options:

- Push to a GitHub repository and enable GitHub Pages (branch: main, folder: /), or
- Deploy to Vercel or Netlify (drag-and-drop the folder or connect the repo).
